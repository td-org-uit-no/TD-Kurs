
# Materiell for introkurs i bruk av terminal

Vi skriver materiell med markdown.

Du kan for eksempel bruke 
[marp](https://marp.app/)
til å bygge slides fra markdown-notatene:

❯ marp terminal-intro.md --pdf-notes -o terminal-intro.pdf
